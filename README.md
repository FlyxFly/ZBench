# Zench

又一个Linux VPS测评脚本


## 说明

感觉这个脚本挺好用的，可以在线查看评测报告，不过原作者的域名已经失效，无法在线查看。我自己搭建了一个在线服务，修改了一下脚本里面对应的链接。

## 脚本命令

如果中文版出现乱码等情况，请换成英文版。

中文版：

    wget -N --no-check-certificate https://github.com/FlyxFly/ZBench/raw/master/ZBench-CN.sh
